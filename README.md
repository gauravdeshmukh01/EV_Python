# Infosys Finacle EdgeVerve PDA Training - Python Programming

This repository is a part of the **Infosys Finacle EdgeVerve PDA Training** program and highlights hands-on learning and practical implementation of **Python Programming**, a powerful and widely used programming language for enterprise application development.

## Repository Overview

This repository includes:
- Core Python programs
- Control flow statements (if, for, while)
- Functions and modular programming
- File handling operations
- Exception handling programs
- Object-Oriented Programming (OOP) concepts
- Practice problems and mini use cases

The structure of this repository aligns with the module-based approach of the training and is organized to aid hands-on learning.

---

## What is Python?

**Python** is a high-level, open-source, and interpreted **programming language** used for software development, automation, data processing, web development, and enterprise systems. It is widely known for:

- **Simple and readable syntax**
- **Fast development speed**
- **Large standard library**
- **Cross-platform compatibility**
- **Strong support for backend and automation development**

This makes Python ideal for use in real-world enterprise applications, including those developed in the Finacle ecosystem.
